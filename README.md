## ARTESAINALS

Este proyecto es un sistema de gestión de inventario diseñado para una salsamentaria, que permite controlar de manera eficiente los productos disponibles, las entradas y salidas de mercancía, y el estado del stock en tiempo real.

El sistema facilita la organización de productos, el registro de ventas y la actualización automática de cantidades, ayudando a reducir pérdidas, evitar desabastecimiento y mejorar la administración del negocio.
